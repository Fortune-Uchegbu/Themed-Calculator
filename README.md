# Calculator App

**Aim:** To practice building a fully functional calculator with clean UI, accurate operations, and theme switching.

**Overview:** A web-based calculator supporting basic arithmetic operations, clear functionality, and a responsive design, with the ability to toggle between multiple visual themes.

**Skills Strengthened:** DOM Manipulation | Event Handling | Arithmetic Logic Implementation | Theme Switching | Responsive Design

**Challenges & Solutions:**  
- *Challenge:* Handling multiple sequential operations without breaking the logic → *Solution:* Implemented a clear state management system for current input, previous input, and selected operator.  
- *Challenge:* Implementing theme switching without affecting functionality → *Solution:* Used CSS variables and JavaScript to toggle themes dynamically.  
- *Challenge:* Ensuring responsive layout for different screen sizes → *Solution:* Used CSS Grid and media queries to adapt the calculator layout for mobile and desktop.

**Key Takeaways:** Learned to manage application state, implement dynamic UI features like theme switching, and ensure a consistent user experience across devices.
